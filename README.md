# Word Generator


## Compilation
gcc wordgen.c -o wordgen


## Usage
Modify the static variables defined in the source to alter the charset used for word generation, and the maximum output length.

```
wordgen

wordgen > wordlist.txt
```

### Console Output
```
...
abcde
bbcde
cbcde
dbcde
ebcde
fbcde
gbcde
...
```
